# Newsletter Back

## Requirements

The server uses Node.js and MongoDB.

## Configuration

The server can be configured via `.env` file:

- `CLIENT_HOST` is the address of the frontend part of the application. Default value is `"localhost"`.
- `CLIENT_PORT` is the port on which the frontend part of the application works. Default value is `5173`
- `DB_HOST` is the address of the MongoDB database. Default value is `"localhost"`.
- `SERVER_PORT` is the port on which the server works. Default value is `3000`
